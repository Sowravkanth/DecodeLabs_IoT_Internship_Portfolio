# Decode Labs IoT Internship Portfolio

> A collection of projects completed during my **IoT Engineering Internship at Decode Labs**, documenting my learning journey in Embedded Systems, Internet of Things (IoT), ESP32 programming, sensor interfacing, and real-world IoT application development.

## 📖 About the Internship

This repository serves as the complete portfolio of my internship at **Decode Labs**, where I am working on hands-on Internet of Things (IoT) projects using microcontrollers, sensors, and embedded programming.

Throughout this internship, I aim to strengthen my practical knowledge of:

- Embedded Systems
- ESP32 Programming
- Arduino IDE
- Sensor Interfacing
- Serial Communication
- IoT System Design
- Hardware Debugging
- Circuit Design
- Technical Documentation
- Git & GitHub Workflow

Every project in this repository is documented with its source code, implementation details, circuit diagrams, project report, and images.
